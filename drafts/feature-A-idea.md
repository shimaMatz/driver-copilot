# Product Overview

トラックCopilotは、2024年4月に厳格化された「430休憩ルール（4時間走行ごとに30分休憩）」に対応するトラックドライバー向けQOL向上・運行支援モバイルアプリ。

## Core Capabilities

1. **430リアルタイム監視** — 連続運転時間と残り走行可能時間を5分単位で可視化（5分ごとの表示更新でよい）
2. **QOLスポット検索** — 大型車駐車可の施設（銭湯・24h飲食店・コインランドリー）を動的にリコメンド
3. **混雑予測・分散休憩誘導** — VICS・走行データに基づきSA/PAの混雑を予測し、空き施設へ誘導

## Target Use Cases

- 帰宅直前に残り時間が30分未満となり強制待機させられるドライバー
- 適切な施設のない区間で休憩場所を探し彷徨うドライバー
- SA/PA満車で駐車できず路肩停車を余儀なくされるドライバー

## Value Proposition

「ルールに縛られる」のではなく「ルールを戦略的に使いこなす」ことで、ドライバーの精神的・身体的負荷を軽減する。法規制への受動的な対応を、能動的な運行最適化に転換する点が差別化ポイント。

## Design Principles

- **1タップ・ビッグボタン** — 運転中・疲労時を考慮した最小タップUI
- **オフライン耐性** — トンネル・山間部でも基本機能（時間計測・地図）を維持
- **リアルタイム精度** — 到着予想時刻の誤差は5分以内
