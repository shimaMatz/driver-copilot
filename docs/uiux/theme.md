| 用途 | フォント | サイズ / 行間 |

## 4. スペーシング・角丸

- **グリッド**: 8px ベース。セクション間は 24〜40px、関連要素は 8〜16px。
- **角丸スケール**: `sm` 6px / `md` 10px / `lg` 14px（ボタン・入力は `md` を既定）。

## 5. モーション

- **原則**: 控えめ。装飾アニメは 200ms 以下、イージング `cubic-bezier(0.4, 0, 0.2, 1)`。
- **継続時間の目安**: hover / focus 150〜200ms、モーダル表示 200〜250ms。

## 6. アクセシビリティ

- **コントラスト**: 本文と背景は WCAG AA 以上を目標。`color-primary` on `color-bg` で不足する場合は `color-primary-hover` または背景側を調整。
- **フォーカス**: キーボード操作時は 2px の可視リング（`color-accent` 系、透明度あり可）。
